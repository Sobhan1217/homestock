# Contributing to HomeStock

Thank you for interest in contributing! Here's how to get started.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/homestock.git`
3. Create a branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Test thoroughly
6. Push your branch and create a Pull Request

## Code Style

- Follow Dart style guide: https://dart.dev/guides/language/effective-dart/style
- Use `flutter analyze` to check for issues
- Run `dart format` to format code

## Pull Request Process

1. Update the README.md with any new features
2. Add tests for new functionality
3. Ensure all tests pass: `flutter test`
4. Request review from maintainers
5. Address feedback and iterate

## Issues

- Check existing issues before creating new ones
- Provide clear reproduction steps
- Include device/OS information
- Add screenshots for UI issues

## Code of Conduct

Be respectful and constructive. This is an open community project.
