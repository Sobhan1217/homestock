# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-17

### Added
- ✨ Complete Flutter app for household inventory management
- 🔐 Firebase Authentication (email/password)
- 📦 Inventory management with stock tracking
- 🛒 Smart shopping list auto-generation
- 📊 Real-time analytics dashboard
- 👨‍👩‍👧‍👦 Family management (stub, ready for Firestore)
- 🌐 Web deployment on EC2
- 📱 Android APK build
- 🍎 iOS app configuration
- 🔄 CI/CD with GitHub Actions
- 🐳 Docker containerization
- 💾 Hive local database

### Features
- Real-time inventory updates
- Category-based organization
- Low stock alerts
- Purchase tracking
- Multi-user support (ready)
- Offline support with Hive

### Tech Stack
- Flutter 3.44+
- Firebase Authentication
- Riverpod for state management
- Hive for local storage
- Docker + Nginx for hosting

### Security
- Firebase secure authentication
- Firestore security rules ready
- No hardcoded secrets
- HTTPS ready

### Known Limitations
- Family sharing requires Firebase billing
- Web-only for now (mobile builds available)
- No push notifications yet

### Coming Soon
